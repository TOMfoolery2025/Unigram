# Unigram Branding

## Project Name
**Unigram** - Community platform for TUM Heilbronn Campus students

## Supabase Project
- **Project Name**: unigram
- **Description**: Community platform for TUM Heilbronn Campus

## Color Scheme

### Primary Colors
- **Primary**: Violet-600 - `hsl(263 70% 50%)` - #8b5cf6
- **Secondary**: Violet-400 - `hsl(262 83% 58%)` - #a78bfa

### Theme
- **Mode**: Dark theme by default
- **Background**: Very dark blue-gray - `hsl(224 71% 4%)`
- **Foreground**: Light gray - `hsl(213 31% 91%)`

### Accent Colors
- **Muted**: Dark gray - `hsl(223 47% 11%)`
- **Border**: Medium gray - `hsl(216 34% 17%)`
- **Ring**: Violet-600 (matches primary)

## Typography
- **Font**: Inter (Google Fonts)
- **Headings**: Bold, primary color (violet)
- **Body**: Regular, foreground color

## Usage

### In Code
```tsx
// Page title
<h1 className="text-4xl font-bold text-primary">Unigram</h1>

// Subtitle
<p className="text-muted-foreground">
  Welcome to Unigram - TUM Heilbronn Campus Community Platform
</p>
```

### In Metadata
```tsx
export const metadata: Metadata = {
  title: "Unigram",
  description: "Community platform for TUM Heilbronn Campus students",
}
```

### In Documentation
- **Full Name**: Unigram

## Brand Voice
- Friendly and welcoming
- Student-focused
- Professional yet approachable
- Community-oriented

- Dark theme creates a modern, focused environment
- Violet accents provide energy and creativity
- Clean, minimal design reduces distractions
